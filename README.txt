# Random Task Generator - Генератор случайных задач

**Автор:** Новокщенов Денис
**Дата:** 2026

## Описание программы

Random Task Generator — это GUI-приложение на Python для генерации случайных задач из предустановленного или пользовательского списка.  
Поддерживается фильтрация по типу задачи (учёба, спорт, работа), сохраняется история всех сгенерированных задач с временными метками в JSON-файл.

## Возможности

- Генерация случайной задачи с отображением на экране
- Фильтрация задач перед генерацией
- Добавление пользовательских задач с проверкой на пустую строку
- Просмотр полной истории генераций
- Сохранение и загрузка истории из `tasks_history.json`
- Очистка истории
